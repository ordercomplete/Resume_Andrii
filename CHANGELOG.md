# CHANGELOG

## [2026-07-08] – Оновлення функціоналу багатомовності та адаптивності

### Додано

- Функція `detectBrowserLanguage()` для автоматичного визначення мови браузера.
- Підсвічування активної мови: кнопки `en` та `uk` отримали клас `active-lang` з відповідним стилем.
- Додано CSS‑правила для розміщення `.language-switcher` праворуч та стилізації активної кнопки.
- Фолбек у випадку помилки завантаження `dictionary.json` (порожній словник).

### Виправлено

- При завантаженні сторінки тепер використовується мова браузера, якщо користувач не обрав мову раніше.
- При помилці `fetch` словника тепер не порушується робота скрипту.
- Після зміни мови оновлюються класи кнопок, що дозволяє підсвічувати поточну мову.

### Адаптивність

- Переконфігуровано розташування кнопок мови у всіх медіа‑запитах, включаючи мобільні розміри, щоб вони залишалися праворуч і не порушували макет.
